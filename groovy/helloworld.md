# Hello World Groovy

- groovyc : javac
- groovy  : java
- .groovy : .java


```groovy
// HelloWorld.groovy
System.out.println("Hello World from Groovy!");
```

```shell
$ groovyc HelloWorld.groovy # creates a .class file
$ groovy  HelloWorld

# compilation step can be skipped
$ groovy HelloWorld.groovy
```
