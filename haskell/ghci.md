# GHCI

-

```shell
$ ghci
$ ghci <filename>      -- start and loads the file

> :q                   -- quit repl

> :l filename          -- load a haskell file

> :r                   -- reread/reload the file

                       -- just a = 1 doesnt work
> let a = 1            -- defining a variable in ghci

> :type variable       -- learn type/signature of a function
> :t (+)
> :t not
```
