# Python Interactive Shell


```shell
# start
$ python3

# exit
>>> ctrl-D
>>> exit()
```

_ -> most recent output being hold in this character

```python
>>> 3 * 5
15
>>> _ + 5
20
```

## Multiline Statements
Tree dots is an indicator of multiline. Enter an empty line to end block.

```python
>>> for i in range(3):
...     print(i)
...
0
1
2
```

## Getting Help

```python
>>> help()
help> keywords
```
