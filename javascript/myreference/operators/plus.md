# + operator

- can be used for addition and concatenation

- if operands are different types, JS converts number to string

- left-to-right associativity

```javascript
1 + 3;               // 4
"1" + "3";           // 13
1 + "3";             // 13

var order = 1 + 2 + "pizza";   // 3pizza
```
